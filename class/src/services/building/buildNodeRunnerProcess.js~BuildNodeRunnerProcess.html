    BuildNodeRunnerProcess

    This service implements both nodemon and watchpack in order to run Node apps while watching and transpiling if necessary.

    public run(executable: string, watchOn: Array, sourcePath: string, executionPath: string, envVars: Object, ignore: Array): Nodemon source

    Run a Node app.

    Params:

    NameTypeAttributeDescription
    executable string

    The app executable.
    watchOn Array

    A list of directories to watch.
    sourcePath string

    The path to the source code of the app. If it doesn't match with executionPath, then the code needs transpilation.
    executionPath string

    The path to where the app is being executed. If it doesn't match with sourcePath, then the code needs transpilation.
    envVars Object
    • optional
    • default: {}

    A dictionary with extra environment variables to send to the process.
    ignore Array
    • optional
    • default: ['*.test.js']

    A list of patterns to ignore on the watch.

    Return:

    Nodemon

    Throw:

    Error

    if the process is already running.

    Error
